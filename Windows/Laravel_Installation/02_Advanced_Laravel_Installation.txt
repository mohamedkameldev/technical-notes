    -Apache24:
        Download apache from here: 
        https://www.apachelounge.com/download/
        Copy instructions from here: 
        https://httpd.apache.org/docs/current/platform/windows.html#down
        
        you can either start it, or deal with it as a service in your machine.
        open cmd as an administrator, and then:
            to run it from the terminal: `httpd.exe` - You can shut down it by pressing Ctrl+C
            to install it as a service: `httpd.exe -k install` - and then start it as a service.
        
    -php: 
        https://windows.php.net/download/ (Thread Safe)  
        Add php to your system variables.
        Edit the System Environment Variables > Environment Variables > System Variables > (Path) > Add the New Path of you php setup.
        php -v
        
        and uncomment mysqli and pdo from php.ini file

    -Apache Configurations:         
        AddHandler application/x-httpd-php .php
        AddType application/x-httpd-php .php .html 
        LoadModule php_module "C:/php/php8apache2_4.dll"
        PHPIniDir "C:/php"
        DirectoryIndex index.php
